使用Python的Pelican来搭建简易个人博客

Pelican的安装与使用

首先安装 Pelican

pip install pelican invoke markdown

在本地仓库文件夹打开终端, 输入下列命令初始化项目:

pelican-quickstart

额外的, 可以在 pelicanconf.py 文件中修改网页的各项配置, 比如指定输出文件夹 OUTPUT_PATH = 'zengyan.github.io/'

将写好的 markdown 笔记放在 content 文件夹即可, 注意文件开头需要包含一些元数据, 如下所示:

Title: 文章标题
Date: 2010-12-03 10:20
Modified: 2010-12-05 19:30
Category: 教程
Tags: 教程
Slug: my-tutorial
Authors: 
Summary:

随后使用下列命令将 markdown 渲染为 HTML , 结果会输出在 output 文件夹内.

pelican content

随后可以用下列命令启动一个本地服务来查看网页:

pelican --listen

发布博客

在Github或者Gitee上新建一个仓库, 注意仓库命名时需要有 .github.io 或者 .gitee.io 后缀, 最简单直接的方式是 name.github.io , 然后把这个仓库克隆到本地.

修改一下 pelicanconf.py 添加以下内容, 然后重新渲染.

SITEURL = 'https://zeng-yan.github.io'
DOMAIN = SITEURL
FEED_DOMAIN = SITEURL
HTTPS = True

把 outpt 文件夹的内容放入仓库中, 将变更上传到远程仓库即可.

Pelican集成Latex公式渲染

安装插件即可

pip install pelican-render-math

Pelican集成Mermaid渲染

在 markdown 库的 extensions 文件夹内新建一个 mermaid.py 文件, 使用 Anaconda 环境的位置大概是: .../anaconda/envs/py310/Lib/site-packages/markdown/extensions , mermaid.py 文件的内容如下:

# -*- coding: utf-8 -*-
"""
Mermaid Extension for Python-Markdown
========================================
Adds mermaid parser (like github-markdown) to standard Python-Markdown code blocks.
Original code Copyright 2018-2020 [Olivier Ruelle].
License: GNU GPLv3
"""

from markdown.extensions import Extension
from markdown.preprocessors import Preprocessor
import re
import string


MermaidRegex = re.compile(r"^(?P<mermaid_sign>[\~\`]){3}[\ \t]*[Mm]ermaid[\ \t]*$")


class MermaidPreprocessor(Preprocessor):
    def run(self, lines):
        new_lines = []
        mermaid_sign = ""
        m_start = None
        m_end = None
        in_mermaid_code = False
        is_mermaid = False
        for line in lines:
            if not in_mermaid_code:
                m_start = MermaidRegex.match(line)
            else:
                m_end = re.match(r"^["+mermaid_sign+"]{3}[\ \t]*$", line)
                if m_end:
                    in_mermaid_code = False
            if m_start:
                in_mermaid_code = True
                mermaid_sign = m_start.group("mermaid_sign")
                if not re.match(r"^[\ \t]*$", old_line):
                    new_lines.append("")
                if not is_mermaid:
                    is_mermaid = True
                new_lines.append('<div class="mermaid">')
                m_start = None
            elif m_end:
                new_lines.append('</div>')
                new_lines.append("")
                m_end = None
            elif in_mermaid_code:
                new_lines.append(line.strip())
            else:
                new_lines.append(line)
            old_line = line
        if is_mermaid:
            new_lines.append('')
            new_lines.append('<script src="https://cdn.jsdelivr.net/npm/mermaid/dist/mermaid.min.js"></script>')
        return new_lines


class MermaidExtension(Extension):
    def extendMarkdown(self, md):
        md.preprocessors.register(MermaidPreprocessor(md), 'mermaid', 35)
        md.registerExtension(self)

def makeExtension(**kwargs):
    return MermaidExtension(**kwargs)

在 pelicanconf.py 中添加以下内容:

# markdown设置
MARKDOWN = {
    'extension_configs': {
        'markdown.extensions.extra': {},
        'markdown.extensions.admonition': {},
        'markdown.extensions.codehilite': {'css_class': 'highlight'},
        'markdown.extensions.meta': {},
        'markdown.extensions.nl2br': {},
        'markdown.extensions.sane_lists': {},
        'markdown.extensions.smarty':{},
        'markdown.extensions.toc': {},
        'markdown.extensions.wikilinks': {},
        'markdown.extensions.mermaid': {},
    },
    'output_format': 'html5',
}
